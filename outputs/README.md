# Outputs Directory

This directory contains all generated outputs from the EHR Embeddings project.

## Structure

- `figures/` - Generated plots, charts, and visualizations
  - Correlation heatmaps
  - Distribution histograms  
  - EDA analysis plots
- `models/` - Trained machine learning models
  - `random_forest_model.joblib` - Trained Random Forest model
- `reports/` - Generated analysis reports and summaries

## Usage

- All generated outputs should be saved to the appropriate subdirectory
- Figures should be saved with descriptive names indicating the analysis type
- Models should include version information and training metadata
- Reports should be dated and include methodology details
