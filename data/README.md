# Data Directory

This directory contains all data files for the EHR Embeddings project.

## Structure

- `raw/` - Raw, unprocessed data files
  - `all_hourly_data.h5` - Main hourly EHR data in HDF5 format (7.2GB)
  - `mimic_legacy.txt` - Legacy MIMIC data file
- `processed/` - Processed and cleaned data files (generated by pipeline)

## Usage

- Place new raw data files in the `raw/` subdirectory
- Processed data will be automatically saved to the `processed/` subdirectory
- Never modify files in `raw/` - always work with copies or processed versions
