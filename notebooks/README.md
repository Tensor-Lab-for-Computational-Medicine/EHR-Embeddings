# Notebooks

This directory contains Jupyter notebooks for the EHR Embeddings project.

## Contents

- `h5_first_5_rows.ipynb` - Exploratory notebook examining the first 5 rows of the H5 data file
- `example_usage.ipynb` - Example usage of the EHR embeddings pipeline (moved from root)

## Usage

Each notebook is self-contained and can be run independently. Make sure to install the required dependencies from the root `requirements.txt` file before running any notebooks.
