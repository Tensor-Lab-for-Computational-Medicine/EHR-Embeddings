# Source Code Directory

This directory contains the main source code for the EHR Embeddings project.

## Structure

- `embeddings.py` - Core embeddings functionality and utilities
- `ml_pipeline.py` - Machine learning pipeline implementation
- `__init__.py` - Package initialization file

## Usage

This is the main package for the EHR embeddings project. Import modules from here:

```python
from src.embeddings import EmbeddingGenerator
from src.ml_pipeline import MLPipeline
```

## Development

- Follow PEP 8 style guidelines
- Add docstrings to all functions and classes
- Include type hints where appropriate
